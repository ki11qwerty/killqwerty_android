# killqwerty_android

Обновление знаний Jetpack Compose

Ознакомиться с основами: композируемые функции, управление состоянием, пользовательский интерфейс.
Изучить документацию и руководства на официальном сайте.
Экран новостей

Определить структуру API для получения новостей.
Реализовать экран со списком новостей с помощью LazyColumn.
Создать экран для отображения детальной информации о новости.
Работа с API

Реализовать получение данных из API (используя Retrofit).
Обрабатывать состояния загрузки и ошибки.
Завершение экрана новостей

Настроить пользовательский интерфейс для экрана новостей.
Протестировать логику приложения и обработку ошибок.
Создание базы данных доходов и расходов с использованием Room

Определить сущности для доходов и расходов.
Реализовать функции добавления, удаления и обновления записей.
Интерфейс для работы с финансами

Создать экран для добавления/удаления записей.
Реализовать сортировку и фильтры по различным критериям.
Подсчет сумм и графики

Реализовать подсчет общих доходов и расходов.
Изучить библиотеку для построения графиков (например, MPAndroidChart) и интегрировать её в приложение.
Практика с Jetpack Compose

Создать тренировочный экран с элементами интерфейса (формы, диалоги и т.д.).
Экспериментировать с различными UI-компонентами и их состояниями.
Рефакторинг и тестирование

Перепроверить код, провести рефакторинг при необходимости.
Добавить юнит-тесты и UI-тесты (JUnit, Compose Test).
Подготовка к собеседованию

Повторить ключевые темы:
Data классы и их использование.
Коллекции и алгоритмы (List, Set, Map).
Coroutines для асинхронного программирования.
Жизненный цикл компонентов (активности, фрагменты).
Архитектуры (MVVM, MVP).
Паттерны проектирования (Singleton, Factory).
Dependency Injection (Dagger, Hilt).
Работа с REST API (Retrofit).
Использование Room для базы данных.
Юнит и UI-тестирование (инструменты и подходы).
